GitHub 또는 GitLab OAuth 앱 생성

OAuth 앱을 사용하면 사용자는 비밀번호를 공유하지 않고도 한 웹 사이트나 서비스의 계정에 대한 액세스 권한을 다른 웹 사이트나 서비스의 계정에 부여할 수 있습니다.

AI Unlimited가 GitHub 또는 GitLab 인증을 사용하여 각 프로젝트 저장소를 관리하고 엔진이 배포될 때마다 저장소에 추가한 공동 작업자로부터 엔진 사용자를 파생할 수 있도록 OAuth 앱을 만듭니다.

  1. GitHub 또는 GitLab 계정에 로그인합니다.

  2. OAuth 앱을 생성합니다. GitHub: OAuth 앱 생성 또는 GitLab: OAuth 앱 생성을 참조하십시오.

    OAuth 앱을 등록하는 동안 다음 URL을 사용합니다.

    • 홈 페이지 URL: http://[ip_or_hostname]:[port]

      노트

      이는 설치 프로세스가 끝나면 받게 되는 관리자 URL입니다.

    • 인증 콜백 URL: [Homepage URL]/auth/github/callback

    GitLab의 경우 인증된 애플리케이션에 정의된 대로 적절한 OAuth 범위를 선택합니다.

3. AI Unlimited를 설정하는 데 필요한 클라이언트 ID 및 클라이언트 암호를 복사하여 보관합니다.

이제 AI Unlimited 설정 준비가 되었습니다.
