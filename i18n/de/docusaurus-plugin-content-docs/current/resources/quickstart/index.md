# Schnellstart

Wenn Sie AI Unlimited einfacher und schneller ausprobieren möchten, verwenden Sie den Schnellstart. 

Dadurch werden der [AI Unlimited-Manager](../../glossary.md#ai-unlimited-manager) und JupyterLab mit dem AI Unlimited-Kernel unter Verwendung von [Docker](https://www.docker.com/) auf Ihrem Computer ausgeführt. 

Der Schnellstart ist für einen einzelnen Benutzer.
