# Cambiar configuración (administrador)

Si es el administrador de AI Unlimited, puede regresar a la configuración de AI Unlimited en el administrador y cambiar cualquier configuración básica, de integración en la nube o de integración de Git.

1. Acceda a la configuración utilizando la misma URL que utilizó para acceder originalmente.

:::note
Recibió la URL cuando instaló el administrador.
:::

2. Inicie sesión en GitHub o GitLab.
3. Después de ser redirigido nuevamente al administrador, seleccione **Configurar**.
3. En cualquier sección, realice cambios y luego seleccione **Actualizar**.<br/>
[Conozca los campos](../install-ai-unlimited/setup-ai-unlimited.md).
4. Inicie sesión para autenticarse.
6. Si habilitó TLS, en su **Perfil**, seleccione **Reiniciar**.<br/>
  AI Unlimited se reinicia con TLS implementado.
