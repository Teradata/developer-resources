## Información general
Este tutorial muestra cómo obtener acceso a una base de datos Teradata ejecutándola en su máquina local. Una vez que termine los pasos, tendrá una base de datos Teradata Vantage Express funcional en su ordenador.

:::note 
A partir de la versión 17.20, Vantage Express incluye los siguientes paquetes de análisis: [Biblioteca de análisis de Vantage](https://docs.teradata.com/r/Enterprise_IntelliFlex_Lake_VMware/Vantage-Analytics-Library-User-Guide), [BYOM](https://docs.teradata.com/r/Enterprise_IntelliFlex_Lake_VMware/Teradata-VantageTM-Bring-Your-Own-Model-User-Guide), [Integración de API con AWS SageMaker](https://docs.teradata.com/r/Enterprise_IntelliFlex_VMware/Teradata-VantageTM-API-Integration-Guide-for-Cloud-Machine-Learning). 
:::
