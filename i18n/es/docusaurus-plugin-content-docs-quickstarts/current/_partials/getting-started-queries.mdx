```sql
CREATE DATABASE HR
AS PERMANENT = 60e6, -- 60MB
    SPOOL = 120e6; -- 120MB
```

* Vamos a crear una tabla de muestra, insertaremos algunos datos y los consultaremos. Primero crearemos una tabla para almacenar la información de los empleados:

```sql
CREATE SET TABLE HR.Employees (
   GlobalID INTEGER,
   FirstName VARCHAR(30),
   LastName VARCHAR(30),
   DateOfBirth DATE FORMAT 'YYYY-MM-DD',
   JoinedDate DATE FORMAT 'YYYY-MM-DD',
   DepartmentCode BYTEINT
)
UNIQUE PRIMARY INDEX ( GlobalID );
```

* Ahora, insertemos un registro:

```sql
INSERT INTO HR.Employees (
   GlobalID,
   FirstName,
   LastName,
   DateOfBirth,
   JoinedDate,
   DepartmentCode
)
VALUES (
   101,
   'Adam',
   'Tworkowski',
   '1980-01-05',
   '2004-08-01',
   01
);
```

* Finalmente, veamos si podemos recuperar los datos:

```sql
SELECT * FROM HR.Employees;
```

Deberías obtener los siguientes resultados:

```sql
GlobalID  FirstName  LastName   DateOfBirth  JoinedDate  DepartmentCode
--------  ---------  ---------- -----------  ----------  --------------
     101  Adam       Tworkowski  1980-01-05  2004-08-01               1
```
